# SyntaxWear

SyntaxWear is a modern and stylish e-commerce website template for selling sneakers online. It features a clean and visually appealing design, with a focus on product presentation and user experience.

## Features

*   **Responsive Design:** The layout adapts to different screen sizes, providing a great user experience on desktops, tablets, and mobile devices.
*   **CSS Grid Layout:** The product grid is built with CSS Grid, allowing for complex and creative layouts.
*   **Modular CSS:** The CSS is organized into a modular structure, making it easy to customize and maintain. It includes:
    *   A CSS reset for cross-browser consistency.
    *   CSS variables for easy theme customization.
    *   A component-based approach for UI elements like the header, footer, and product cards.
*   **Hero Section:** A prominent hero section to showcase featured products or promotions.
*   **Product Categories:** A section to display different product categories with visually appealing cards.
*   **Newsletter Signup:** A newsletter signup form in the footer to capture user emails.
*   **Social Media Integration:** Links to social media profiles in the footer.

## Technologies Used

*   **HTML5:** For the structure and content of the website.
*   **CSS3:** For styling the website, using modern features like CSS Grid and Flexbox.
*   **Google Fonts:** For custom web fonts.

## How to Run

To run this project, you can simply open the `index.html` file in your web browser.

1.  Clone this repository:
    ```bash
    git clone https://github.com/cristian-souza/ecommerce-syntaxwear.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd ecommerce-syntaxwear
    ```
3.  Open the `index.html` file in your browser.

---

This is a template project and does not include any backend functionality or JavaScript for dynamic interactions. It serves as a solid starting point for building a full-fledged e-commerce website.
